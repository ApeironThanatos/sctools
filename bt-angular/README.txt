Welcome to a SP-800-53 online editor.

Dependencies:
	- nodejs
	- bower
	- grunt-cli
	- npm

Steps to run:
	1: Execute 'npm install' in the local directory that contains 'package.json'
	2: Execute 'grunt' from the bt-angular directory to run the server.
	3: Type 'localhost:8004' into the search field of your browser
