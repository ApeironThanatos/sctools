Welcome to a SP-800-53 online editor.

Dependencies:
	- nodejs
	- bower
	- npm

Steps to run:
	1: execute npm install in the local directory that contains 'package.json'
	2: execute bower install in the local directory that contains 'bower.json'
	3: Execute 'grunt' from the bt-angular directory to run the server.
	4: type localhost:8004 into the search field of your browser
